## 事業内容

### 主要サービス

1. **ITプロジェクトマネジメント代行・支援**
   - 大規模ITプロジェクトの推進・管理
   - PMO（プロジェクトマネジメントオフィス）支援

2. **ITプロジェクト技術支援**
   - 最新IT技術を活用した開発支援
   - 技術的な課題解決

3. **ビジネスITコンサルティング**
   - 事業計画コンサルティング
   - IT戦略立案

4. **小規模開発受託**
   - カスタムソフトウェア開発

### 主要クライアント

- 銀行・金融機関
- 通販企業
- 金融系事務代行企業
- LINE Xenesis
- IIJ（インターネットイニシアティブ）

## 企業の特徴

### 理念・強み

- **「技術者であり続ける」** を掲げ、マネジメントとIT技術の両面からプロジェクトを支援
- マネジメントスキルと最新ITスキルを兼ね備えた技術者がクライアントと共にプロジェクトを推進
- 3つのコアコンピタンス：
  1. 事業構築力
  2. IT技術力
  3. マネジメント力

### 代表者経歴

- 1991年：大手ITベンダーに入社
- 証券・金融系を中心に情報システム開発、プロジェクトマネジメントを多数経験
- 2008年：退職後、プライム・ブレインズを設立
- **保有資格**：
  - 情報処理安全確保支援士
  - 高度情報処理技術者（プロジェクトマネージャ、ITストラテジスト、システム監査、アプリケーションエンジニア）

## 成長・実績

- 社長一人での創業から15期目を迎え、従業員50名超に成長
- 当初はユーザー企業向けPM支援からスタート
- 現在はITプロジェクトマネジメント、IT技術、ITコンサルティング、開発まで領域を拡大
- 暗号資産送金規制対応などコンプライアンス関連プロジェクトの実績あり

- 新卒・中途採用を積極的に実施

## 調査所見

プライム・ブレインズ社は、ITプロジェクトマネジメントに特化したコンサルティング企業。代表者の豊富な金融系システム開発経験を基盤に、「技術者であり続ける」という理念のもと、PMだけでなく技術面からもプロジェクトを支援する点が特徴的。大手金融機関や通販企業などを顧客に持ち、着実に成長を続けている。
